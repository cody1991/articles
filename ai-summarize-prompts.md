## Prompt 1: 金渐层 - 投资原则提取

### 用途
从投资文章中系统化提取投资原则、心法和可复用的方法论，归纳到 5 个维度。

### 完整 Prompt

```
你是一位资深投资分析师。现在我要你分析以下投资文章，系统化地提取其中的投资原则和智慧。

【分析框架】
请从以下 5 个维度深度分析该文章，提取可复用的投资原则：

1. 📊 **择时选股**（市场判断和选股方法论）
   - 主要观点：作者如何判断市场方向？
   - 具体选股逻辑：如何选择标的？
   - 可复用原则：这套逻辑适用于什么场景？

2. 💼 **风险管理**（仓位控制、止损、分批操作）
   - 仓位配置：文中提到的仓位大小和分配？
   - 风险控制：遇到下跌如何处理？
   - 关键指标：什么时候加仓、减仓？

3. 🧠 **心态建设**（心理素质、执行纪律、长期思维）
   - 核心心态：作者强调的心理关键点是什么？
   - 常见陷阱：文中警告的哪些错误思维？
   - 执行纪律：坚持什么、拒绝什么？

4. 💡 **方法论**（具体操作步骤、分析工具、决策流程）
   - 操作步骤：作者如何一步步执行的？
   - 分析工具：用了什么方法论或指标？
   - 复现难度：我学会这套方法需要什么前提条件？

5. ⚠️ **风险提示**（作者指出的风险、黑天鹅事件、需要警惕的点）
   - 明确风险：文中提到的具体风险是什么？
   - 应对方案：遇到这些风险如何应对？
   - 底线设定：绝对不能做什么？

【输出格式】
请按照以下 JSON 结构返回分析结果：

{
  "文章标题": "...",
  "发布日期": "...",
  "核心观点": "用一句话概括文章最重要的观点",
  "重要程度": "★★★★★（5 星制，表示这个文章的启发程度）",
  
  "择时选股": {
    "主要观点": "...",
    "具体选股逻辑": "...",
    "可复用原则": "..."
  },
  
  "风险管理": {
    "仓位配置": "...",
    "风险控制": "...",
    "关键指标": "..."
  },
  
  "心态建设": {
    "核心心态": "...",
    "常见陷阱": "...",
    "执行纪律": "..."
  },
  
  "方法论": {
    "操作步骤": "...",
    "分析工具": "...",
    "复现难度": "..." 
  },
  
  "风险提示": {
    "明确风险": "...",
    "应对方案": "...",
    "底线设定": "..."
  },
  
  "个人笔记": "阅读这篇文章最有启发的 1-2 点是什么？",
  
  "标签": ["标签1", "标签2", "标签3"]（比如：["抄底机会", "心态建设", "仓位管理"]）
}

【分析要求】
- 严格按照上述 5 个维度深度分析，不要遗漏
- 提取的原则必须是「可复用的」，即便换个市场环境也适用
- 不要空泛地转述原文，要进行系统化的归纳提炼
- 如果某个维度文中没有涉及，请填写 "文中未提及"
- 用专业投资术语进行表述

【现在，请分析以下文章】

{文章内容粘贴在这里}
```

## Prompt 2: 市场复盘 - 时序数据提取

### 用途
从每日市场复盘中提取结构化数据，便于后续验证预测准确性和趋势分析。

### 完整 Prompt

```
你是一位金融数据分析师。现在我要你从以下市场复盘文章中提取结构化的时序数据和交易信号。

【提取维度】

1. 📈 **市场热点和关键资产**
   - 资产名称
   - 当日表现（涨跌幅、价格）
   - 强度评级（强势/正常/弱势）
   - 背后驱动力
   
2. 🔮 **作者预判和信号**
   - 预测内容（会涨还是会跌？）
   - 预测强度（强烈看好/谨慎看好/观望/谨慎看空/强烈看空）
   - 核心理由（基本面/技术面/情绪）
   - 关键阈值（会在哪个价位转向？）
   
3. ⚠️ **风险提示**
   - 风险类型
   - 风险等级（低/中/高/极高）
   - 应对策略
   
4. 💰 **交易机会**
   - 机会类型（布局/加仓/减仓/观望）
   - 推荐价位
   - 风险收益比
   - 前置条件
   
5. 🌍 **外部因素**
   - 重要事件（FOMC、财报、经济数据等）
```

## Prompt 3: 口罩哥 - 美股操作策略分析

### 用途
从美股深度分析文章中提取操作策略、时机判断和衍生品使用技巧，支持期权等复杂交易。

### 完整 Prompt

```
你是一位美股交易策略分析师。现在我要你分析以下美股投资文章，提取其中的交易策略和操作智慧。

【分析框架】
请从以下 5 个维度深度分析该文章：

1. 📊 **标的分析**（目标股票或资产的基本面与技术面判断）
   - 目标标的：关注哪只股票或资产？
   - 基本面判断：为什么选这个标的？（财报、增长、竞争力等）
   - 技术面信号：价格位置、支撑阻力、趋势判断
   - 估值观点：价格是否被低估/高估？

2. 💰 **交易策略**（具体的交易方法、工具、执行步骤）
   - 策略类型：期权/现货/组合策略？（如 Leap Call、Cash Secured Strangle 等）
   - 策略逻辑：为什么选这个策略？
   - 具体执行：买卖价位、数量、执行顺序
   - 预期收益：目标收益率或收益空间

3. ⏰ **时机把握**（何时操作、何时止盈止损、持仓周期）
   - 建仓时机：什么时候应该启动？（关键事件、技术面突破等）
   - 持仓周期：计划持多久？
   - 止盈点：在哪个位置减仓/平仓？
   - 止损点：风险底线是多少？

4. 🎯 **事件驱动**（宏观数据、财报、政策等关键事件的影响）
   - 关键事件：CPI、FOMC、财报、关税等
   - 事件时间：什么时候发生？
   - 影响分析：对目标标的的直接影响是什么？
   - 操作建议：事件发生前后如何调整策略？

5. ⚠️ **风险管理**（仓位大小、风险对冲、黑天鹅防范）
   - 仓位配置：投入多少资金？占总资产的比例？
   - 风险对冲：用什么工具对冲风险？
   - 可能风险：文中提到或隐含的风险是什么？
   - 应急预案：如果判断错误如何处理？

【输出格式】
请按照以下 JSON 结构返回分析结果：

{
  "文章标题": "...",
  "发布日期": "...",
  "核心观点": "用一句话概括文章最重要的交易观点",
  "重要程度": "★★★★★（5 星制）",
  
  "标的分析": {
    "目标标的": "...",
    "基本面判断": "...",
    "技术面信号": "...",
    "估值观点": "..."
  },
  
  "交易策略": {
    "策略类型": "...",
    "策略逻辑": "...",
    "具体执行": "...",
    "预期收益": "..."
  },
  
  "时机把握": {
    "建仓时机": "...",
    "持仓周期": "...",
    "止盈点": "...",
    "止损点": "..."
  },
  
  "事件驱动": {
    "关键事件": "...",
    "事件时间": "...",
    "影响分析": "...",
    "操作建议": "..."
  },
  
  "风险管理": {
    "仓位配置": "...",
    "风险对冲": "...",
    "可能风险": "...",
    "应急预案": "..."
  },
  
  "个人笔记": "这篇文章最有启发的交易技巧是什么？",
  
  "标签": ["标签1", "标签2", "标签3"]（比如：["期权策略", "财报交易", "止盈设定"]）
}

【分析要求】
- 严格按照上述 5 个维度深度分析，不要遗漏
- 对期权等衍生品的描述要精确（策略名称、执行细节）
- 提取的策略必须是「可复现的」，即便换个股票也能应用
- 特别关注「时机」和「风险」两个维度，这是交易成功的关键
- 如果某个维度文中没有涉及，请填写 "文中未提及"
- 用专业交易术语进行表述

【现在，请分析以下文章】

{文章内容粘贴在这里}
```

## 高级用法

### 4️⃣ Prompt 4: 跨文章聚类（找出同一主题的多篇文章）

当你有多篇文章的分析结果后，可以用这个 Prompt 进行聚类：

```
你是一位投资研究员。我现在有多篇文章的分析结果（JSON 格式）。

请根据以下维度进行聚类分析：

【聚类维度】
1. 核心观点相似性：哪些文章讨论的是同一个投资主题？
2. 时间序列：这个主题是如何随时间演进的？
3. 预测准确性：作者的预测是否验证了前面的观点？
4. 观点演变：作者对某个资产的看法是否有变化？

【输出格式】
{
  "聚类主题": [
    {
      "主题名": "抄底科技股",
      "相关文章": ["文章标题1", "文章标题2", "文章标题3"],
      "时间跨度": "2024-09 到 2025-01",
      "观点演变": "从等待→开始布局→持续加仓→准备减仓",
      "预测准确率": "80%",
      "启发": "..."
    },
    // ... 其他主题
  ]
}

{粘贴多篇文章的分析结果 JSON}
```

### 5️⃣ Prompt 5: 预测验证（检验作者的预测是否成真）

```
你是一位投资业绩审计员。请根据以下信息验证作者的预测准确性：

【验证信息】
- 原始预测（JSON）：{粘贴之前的预判数据}
- 实际结果数据：{粘贴后续市场表现}

【验证维度】
1. 方向判断是否正确（涨/跌/横盘）
2. 幅度预估是否准确（±5%以内为准确）
3. 时间判断是否准确（时间框架是否符合）
4. 原因分析是否经得起推敲

【输出格式】
{
  "预测编号": "预测1",
  "原始预测": "...",
  "实际结果": "...",
  "判断准确": "✓ 正确 / ✗ 错误",
  "准确度": "95%（具体说明）",
  "启发": "这个预测验证了什么投资原则？"
}
```
