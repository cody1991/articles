# 自由以后想做的事儿居然提前做完了大半

**发布时间**: 2020-10-13 06:59:00

**原文链接**: [http://mp.weixin.qq.com/s?__biz=MzUzNjE3NzQ3Nw==&mid=2247487104&idx=1&sn=60163e4b5b5918720964e8a107482d14&chksm=fafb70aacd8cf9bcb45e62dbf73d5e21cf63e84ed70486215e97d8858ffe592f9d549f14299e#rd](http://mp.weixin.qq.com/s?__biz=MzUzNjE3NzQ3Nw==&mid=2247487104&idx=1&sn=60163e4b5b5918720964e8a107482d14&chksm=fafb70aacd8cf9bcb45e62dbf73d5e21cf63e84ed70486215e97d8858ffe592f9d549f14299e#rd)

先给你们讲一个从 reddit 财务自由社区看到的故事。

题主分享自己的理财经历，典型的先知先觉型，“虽然不知道未来向往的生活是什么样的，但是我相信储蓄要趁早，所以早早开始理财”。从 23 岁就已经开始投资理财了。

后来他有幸交往了一位同样存钱小能手的女朋友，一顿安利理财和财务自由，俩人一拍即合，一起努力奋斗财务自由。

看起来非常励志的故事是不是？

2 年半以后，俩人分手了。后来男生写下了这篇帖子（原帖的链接我放在文末「阅读原文」中）：

> 我们一直在考虑，自己未来要如何如何，财务自由以后要如何如何，却很少花时间考虑现在应该怎样。
> 
> 我们非常努力地工作，执着地赚钱、理财，却很少认真享受当下，解决现在的问题。
> 
> **I built my savings but never built my life……**

我们总是寄希望于，攒下多少钱就能如何如何。却很少考虑这个非常现实的问题：

  * 攒下这笔钱，或者财务自由了，这个问题真的就解决了吗？

  * 解决这个问题，真的非要这么多钱不可吗？

现实经常是，大部分人的钱永远都攒不够，所以问题一拖再拖。一小部分人虽然攒够了钱，结果却发现问题依旧。

当然，我不是说攒钱没用。如果有人说「有的快乐钱买不到」，我第一个就会跳出来反驳「**金钱买不来的快乐，你没钱难道就更容易了吗？** 」。

对于大部分人来说，财务是非常明显的短板，**至少从 0 到几百万，对于生活的改善非常明显** 。

但我们不能寄希望于通过钱解决全部问题，甚至认为，唯有钱才能解决问题，比如「等我有 xxx 元以后……」。

……

老读者可能有印象，去年我调查过一次「[大伙自由以后想干啥的](https://mp.weixin.qq.com/s?__biz=MzUzNjE3NzQ3Nw==&mid=2247484107&idx=1&sn=b1c860ace57601b49a8623800bd4cf0c&scene=21#wechat_redirect)」。当时我写了几个：

  * 让自己有理由像我家猫一样懒，而且像他们一样懒的心安理得；

  * 感受一把肥宅生活，赖在家里打游戏，叫外卖；

  * 继续写东西分享，会花更多时间和精力。

很多人都提到的学习、读书、休假这几个方面，我也差不多。

现在回忆起来，一年多过去了我还是没有自由，但是想做的事儿居然大部分都做完了，**和我之前想得很不一样。**

我发现[ 并不需要自由，也可以花更多精力用来写作分享](https://mp.weixin.qq.com/s?__biz=MzUzNjE3NzQ3Nw==&mid=2247485107&idx=1&sn=397eb78a96726ae52ebf0daba97d07d6&scene=21#wechat_redirect)。本以为财务自由才能专心写东西，结果没想到反而是写作副业帮助财务自由更近一步。

我发现并不需要自由，也可以享受几个懒散的周末和假期。本以为努力工作换来财务自由才能放心休息，结果没想到反而是[好好休息才更有精力努力工作](https://mp.weixin.qq.com/s?__biz=MzUzNjE3NzQ3Nw==&mid=2247484874&idx=1&sn=906c5a5145110ca1a9b66728d0f19168&scene=21#wechat_redirect)。（而且像猫一样懒的生活并不舒服，觉睡多了真头疼。）

我发现并不需要自由，也可以大批量读书，甚至**工作日阅读效率比周末更高** 。本以为是有时间才能读书，结果没想到反而是[书读得越多，闲暇时间更多](https://mp.weixin.qq.com/s?__biz=MzUzNjE3NzQ3Nw==&mid=2247486483&idx=1&sn=b312362bfb6a90ad19cb2ef4645e93dc&scene=21#wechat_redirect)。

自由的生活并不用拖到财务自由以后，之前一位朋友的总结挺在理的：

> 努力获得心灵自由，如果已经巨有钱了再去开始研究心灵自由基本上已经晚了，所以要在还没钱的时候提前开始。

我给自己定的 500 万财务自由目标，经常有人问我，凭你的赚钱能力为啥不定 1000 万呢？等你定了 1000 万的目标，又会有人问你为什么不是 2000 万？

但却很少有人问，**你想攒下这笔钱，究竟为了解决什么问题？** 那么，解决这个问题，究竟需要多少钱，真的需要那么多吗？

我们生活中的大部分问题其实都有现成的解决方案：

  * 想要更多的时间精力来做有意义的事儿，学学任务管理和精力管理。

  * 想要更强的抗风险能力、不焦虑未来的医疗开支，早早配置几份保险。

  * 想要更多时间陪伴家人，一个电话或者休个假就能解决。

  * 想要减少对财富和通胀的焦虑，开始计划、开始行动，不需要账户里有很多钱，只要看着钱在变多、增值，焦虑就能解决。

这些问题真得等到有 XXX 元或者财务自由以后才能解决吗，那生活得多么辛苦？
