# 辞职 3 个月，找回了成长的感觉

**发布时间**: 2023-08-16 10:15:38

---

很长一段时间我都抱有这样的看法：

  * 一件事儿如果我们总觉得「得等自由才能做」，那大概率真自由了也不会做。

  * 如果你想做的事很多都和成长有关，上班这样有“他律”的规律生活状态会更好。就像工作日通勤可以读不少书，周末反而看不了几页。

看到这样的例子越多，越是深信不疑，直到最近**我遇到了反例** 。有位伙伴攒够钱辞职 3 个月了，他真的开始实践自己说了很久的事儿，而且状态越来越好。

以前聊天，聊的通常都是吃什么、玩什么、买什么，提起工作大多是吐槽。现在会更多聊起在做什么，未来有什么打算，有什么目标。

要不知道的，可能还以为他现在才是在工作 😂

我很不解，「既然这些事你很喜欢，为啥过去这么久都找不到一点机会？这不合理吧？」

结果他居然搬出了 MBTI 十六人格 ![](images/2023-08-16_辞职 3 个月，找回了成长的感觉_img0.png) 但听完以后觉得这个回答还挺在理的，是我没想过的另一种视角。

……

MBTI 人格中有四个评价维度：

  * I/E 内向/外向

  * N/S 直觉型关注未来目标/现实型关注当前状况

  * T/F 理性/感性

  * J/P 执行力高、内耗小/喜欢变化、犹豫分心

他说觉得我是个内耗很小的人，如果我想做一件事儿，基本就只会考虑如何做、怎么做，很少会去纠结这件事能不能做、别人怎么看、为什么（INTJ，确实是我一贯的做事方式）

但另一种性格类型 P，他们更有创造力，但也内耗更大。「**假如我兜里有 10 快钱，当我花到 5 快钱的时候，就已经相当于兜里没有钱了。资源匮乏导致的纠结和内耗会让我僵在那里，什么也做不了** 」。时间也是如此。

所以 P 型人格通常空闲时间越多，效率才越高。

小伙伴还举了一个花钱的例子——以前他给自己设的预算目标很低，逼着自己攒钱，结果总是超支。最后受不了直接**把预算加了一倍，结果每月开销反而比以前还低了** 。

我看问题更接近《俭省》中的理念「资源适度匮乏有利于创新，如果资源不限量就没有动力寻找更高效的解决办法」。就像现实中，发达地区的自然资源很多并不丰富，反而是自然资源丰富的地区前期卖资源赚钱太容易了，没啥创新动力。

但小伙伴觉得自己要在一个资源充足、不用瞻前顾后的状态里，才能更好思考自己该做什么。有点像 HBO 的经营理念「我们之所以成功，是因为我们不会因为一部剧的成败而影响公司收入，所以能够放手创新和尝试」。

两句话单独看都很有道理，但思路居然是相反的。

聊完以后最大的收获——我不会改变自己做事的方式，但对别人的选择多了一点理解。关键可能不在这些理念、方法，而是要先清楚自己到底是哪类人。**认清自己才是最重要，也是最难的事儿** 。

……

最后聊到成长，我说感觉他状态特别好。

「很多人说辞职以后会无聊、空虚、没有价值感，我是完全没有这种感觉。且不管以后收入如何，光是这种找回自己不断成长的感觉，我就觉得非常值了。」

我很认同。

「成长」有多值钱？想到万 sir 分享过一本去年的书《别相信直觉：运用数据获取你真正想要的生活》。里面讨论了一个问题：**相亲时找什么样的对象未来更容易幸福** ？

直觉的标准通常是相貌、身高、种族、职业。但研究发现它们都不是，尤其是作者对职业这一项的看法——事业心只能说明这个人的利己属性，但让他人幸福需要的是利他属性。

数据得出的四个标准是：

  1. 生活满意度。如果他原本就是个幸福快乐的人，多半也能让你幸福（所以也有人说幸福其实是一种能力）

  2. 安全依恋风格，相互依恋但又有安全感相互信任；

  3. 尽责性，也就是责任心，办事靠谱、为人自律；

  4. 成长思维模式，愿意不断成长提高自己；

成长思维也是影响幸福感和幸福关系的决定性因素。如果我们赚钱的目标是让自己开心幸福，那成长就至少值 1/4 个财务自由了。
